The composer turns a request into a structured packet an agent can act on.

## Packet fields

- Goal
- Context
- Constraints
- Allowed actions
- Approval gates
- Acceptance criteria
- Proof required

## Example

> Restore the full website architecture from the reference, preserve every page and docs route, add the sticky intent-scroll component near the top, rewrite copy for Niyatna, and prove it with build output plus route checks.

That is an intent. “Make it better” is not.
