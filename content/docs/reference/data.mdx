Proof should be findable after the session ends.

## Proof artifacts

- patches and diffs
- screenshots
- build logs
- test output
- browser route checks
- source extracts
- approval notes

## Storage principle

Keep sensitive data private. Keep proof attached to the task. Never expose secrets inside public logs, screenshots, docs, or memory.
