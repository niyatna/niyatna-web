---
title: Keyboard shortcuts
description: "Shortcuts for navigating intent, proof, agents, and review states."
icon: Keyboard
---

Shortcuts should move the reviewer through the work without hiding context.

| Action | Shortcut |
| --- | --- |
| New intent | `Mod+N` |
| Open command palette | `Mod+K` |
| Focus intent rail | `Mod+I` |
| Focus proof panel | `Mod+P` |
| Toggle agent lane | `Mod+J` |
| Mark approval needed | `Mod+Shift+A` |
| Open docs | `Mod+/` |

Exact bindings can change by product surface, but the principle stays the same: fast navigation, no loss of proof.
