Tool integration exists so work can return evidence automatically.

## Event shape

A tool should report:

- intent id
- action performed
- status
- evidence path or URL
- error or blocker if any
- whether human approval is needed

## Why events matter

Without structured events, agent work becomes a pile of logs. With structured events, the proof layer can show what happened and what decision remains.
