The visual language of Niyatna HQ is designed around one core mission: **Empowering Human Intent**. 

To maintain high strategic focus and prevent dashboard fatigue, the Command Room interface uses a silent-premium layout: dark graphite, matte porcelain, soft line-waves, and minimal layout noise.

## Design Standards

*   **Matte Palette:** Deep graphite backgrounds, soft gray text, and off-white/porcelain highlight surfaces. No neon AI glows.
*   **Structured Layouts:** Spatially organized workspaces presenting intent, logs, and proof in clean, scannable hierarchies.
*   **Micro-Animations:** Fluid line-wave animations indicating agent execution without visual distraction.
*   **Clear Decision Gates:** Distinct layouts highlighting actions awaiting veto or approval, ensuring security options are always clear.

## Visual Trust

Trust is visual. A chaotic dashboard makes it difficult for operators to understand what an agent did. Niyatna's visual system remains calm, inspectable, and silent, providing a high-trust interface for reviewing critical operations.
