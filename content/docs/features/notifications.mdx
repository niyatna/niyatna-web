Notifications exist to return attention at the right moment.

A useful notification says what happened, what proof exists, and whether a decision is needed. It does not spam every internal step.

## Event types

- Agent started
- Agent needs approval
- Verification passed or failed
- Proof bundle ready
- Unexpected blocker

## Good notification shape

`intent → status → evidence → decision needed`
